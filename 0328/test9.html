Display 속성

Display속성은 각각의 요소를 어떻게 보여줄지를 결정한다.

속성처리값? visibility(보여주기 속성- 기본값) / hidden(감추기) / inherit(부모요소 값을 상속받는다)

none : 보여주지 않을때
inline : 인라인상태(자신의 고유영역만 확보) - 단어의 의미로 해석?
block : 박스영역을 차지하는 속성 - 문단의 의미로 해석?
inline-block : 두 속성을 합친 것
